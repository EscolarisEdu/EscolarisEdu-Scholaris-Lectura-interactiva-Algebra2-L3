Unidad 1 · Una señal para elegir la ruta

Ecuaciones cuadráticas con soluciones complejas

Una raíz negativa no detiene el proceso. El discriminante te avisa cuándo debes usar números complejos.

Meta: resolver una ecuación cuadrática con coeficientes reales cuando sus soluciones son complejas.
Δb² − 4ac
Si Δ < 0, la ruta continúa con i.
1 · Prepara la ecuación

Primero identifica a, b y c

La fórmula cuadrática funciona cuando la ecuación está escrita como ax² + bx + c = 0.

x² + 4x + 13 = 0

Toca cada tarjeta para descubrir el coeficiente.

Precisión: conserva el signo de cada coeficiente. Un signo incorrecto cambia el discriminante.
2 · Lee la señal

El signo de Δ anticipa el tipo de solución

Toca las tres señales. No necesitas resolver toda la ecuación para hacer esta predicción.

Revela las tres señales para completar el mapa.
3 · Calcula antes de simplificar

El discriminante decide la ruta

Para x² + 4x + 13 = 0, sustituye a = 1, b = 4 y c = 13.

Δ = b² − 4ac
Δ = 4² − 4(1)(13)
Δ = 16 − 52
Δ = −36
Observa los signos y revela una línea a la vez.
4 · Continúa con i

Una raíz negativa se transforma

√(−36) = √36 · √(−1) = 6i
Evita una duplicación: no escribas ± dentro de la raíz y otro ± fuera. La fórmula ya incluye el símbolo que separa las dos rutas.
5 · Usa la fórmula completa

Conserva ± hasta el final

x = (-b ± √Δ) / 2a
x = (−4 ± √(−36)) / 2
x = (−4 ± 6i) / 2
x = −2 ± 3i
x₁ = −2 + 3i
±
x₂ = −2 − 3i
6 · Comprueba una solución

Sustituir confirma que el resultado funciona

Probaremos una de las dos soluciones

x = −2 + 3i

La ecuación original es:

x² + 4x + 13 = 0

Revela la sustitución línea por línea.

(−2 + 3i)² + 4(−2 + 3i) + 13
= (−5 − 12i) + (−8 + 12i) + 13
= (−5 − 8 + 13) + (−12i + 12i)
= 0 + 0i = 0 ✓
7 · Verificación rápida

Clasifica antes de resolver

Usa Δ = b² − 4ac para responder. Recibirás retroalimentación inmediata.

1. En x² + 2x + 5 = 0, Δ = 4 − 20 = −16. ¿Qué tipo de soluciones tiene?
2. ¿Cómo se simplifica √(−4)?
3. ¿De dónde sale el símbolo ± que produce las dos soluciones?
Responde las tres preguntas.
✓
8 · Ruta completada

El discriminante no detiene la solución: orienta el siguiente paso

1. OrganizaEscribe ax² + bx + c = 0.
2. InterpretaSi Δ < 0, usa i.
3. CompruebaSustituye una solución.
x² + 4x + 13 = 0  →  x = −2 ± 3i
